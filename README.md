# TEAL d.o.o. - Spletna stran

## O projektu
Spletna stran za podjetje TEAL d.o.o. (Lasko), lastnika platforme Outletko.si.
Oblikovana po vzoru pc-pro.si z vsebino iz Outletko.si.

## Struktura
- index.html   - Glavna HTML datoteka (545 vrstic)
- style.css    - CSS datoteka (responsive, mobilno prilagojena)
- images/
  - hero.jpg       - Hero banner slika
  - about.jpg      - Slika za "O nas" sekcijo
  - refurbished.jpg - Slika za outlet/obnovljeno sekcijo

## Sekcije
1. Top bar         - Kontaktni podatki + gumb Outletko.si
2. Header/Nav      - Logo, navigacija, hamburger meni (mobil)
3. Hero            - Polnoekranski banner z statistikami
4. USP bar         - 4 prednosti podjetja
5. Ponudba         - 8 kategorij produktov z linki na Outletko.si
6. O nas           - Zgodba podjetja od 1991
7. Outlet          - Highlight za obnovljeno/refurbished opremo
8. Storitve        - 4 servisne storitve
9. CTA Banner      - Gumb na Outletko.si
10. Kontakt        - 2 lokaciji (Lasko, Celje)
11. Footer         - Navigacija, kontakt, link Outletko.si
12. Floating CTA   - Plavajoci gumb Outletko.si
